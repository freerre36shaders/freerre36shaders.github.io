Changelogs for RRe36's Shaders

Here you can find the changelogs of the latest versions of all of RRe36's shaders.

FastPBR Shader Changelog

Version: v1.1 (Latest)
Release date: June 13, 2023

Changelog:

  • Fixed weather particle rendering
  • Removed vanilla entity shadows in the nether
  • Fixed incorrectly rendering volumetric fog in some indoor scenes
  • Added block mappings for Minecraft 1.20
  • Added support for color management capability of Iris 1.6.4
  • Improved motion blur
  • Fixed missing glow on dim emitters
  • Added proper moon phase handling
    • Moon phases now determine the brightness of the moonlight

Kappa Shader Changelog

Release date: June 12, 2023

Changelog:

  • Fixed strange emission behavior when using resource pack reflections

    • Including new blocks of Minecraft 1.20

  • Improved motion blur
  • Added support for color management capability of Iris 1.6.4
  • Added proper handling of moon phases

Soft Voxels Lite Changelog

Version: v2.0 (Latest)
Release date: June 12, 2023

Changelog:

  • Fixed performance loss by SSAO being calculated for areas outside the view
  • Fixed depth of field focus plane visualization
  • Added block mappings for Minecraft 1.19 and 1.20
    • Including various new emitter mappings, especially for sculk blocks
  • Improved TAAU
    • Now at 75% by default to compensate for edge sharpness of the lighting compared to the full version (the latter uses spherical harmonics to resolve this)
    • Also exposed various settings to control TAAU behavior
  • Fixed various minor NaNs in some scenes
  • Improved distant lightmap fallback for emission
  • Raised default image sharpening strength to 0.7
  • Added support for color management capability of Iris 1.6.4
    • Alternatively a setting to choose the display color space when on OptiFine
  • Added a setting to toggle the purkinje effect
  • Color grading changes through RRe36's custom film stock emulation
  • Added support for the nether
  • Adjusted atmosphere parameters
  • Adjusted auto exposure behavior
  • Improved motion blur
  • Fixed lighting on cacti
