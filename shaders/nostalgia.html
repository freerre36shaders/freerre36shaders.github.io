Nostalgia Shader

What is the Nostalgia Shader?

This shaderpack is intended to reproduce the look and feel of the "old" shaderpacks from back when the shaders mod was new. However, it also builds ontop of that with new features and superior performance. Accidentally, it also has some similarities to the now cancelled "Super Duper Graphics Pack." If you like the look of this pack but want something more advanced, you can check out NostalgiaVX, which features world-space path-traced lighting using voxels.
