Guide for Installing Shaders

This is a guide which teaches you how to install the shaders you can find on RRe36's Shaders for Free.

How do I install these shaderpacks?

  1. Install OptiFine or Iris (many guides can be found online). Most of the shaderpacks here work with both, but FastPBR, NostalgiaVX, and Soft Voxels require Iris. Make sure to read the file names.

  2. Download any one of the shaderpacks you can find on this website.

  3. Type "%appdata%" into the Windows search bar and then press enter.

  4. Navigate to ".minecraft" and then to "shaderpacks".

  5. Drag the shaderpack you downloaded (usually in the "Downloads" folder) into the "shaderpacks" folder you just navigated to.

  6. Open Minecraft with Optifine/Iris, and enable the shaderpack in the "Video Settings" then "Shaderpacks" options.

  7. Enjoy! Many of these shaderpacks have high performance costs, so make sure to use the one with the style and performance configuration you like. It is always recommended to experiment with the "Shaderpack Settings" to get the experience you want.
